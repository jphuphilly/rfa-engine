# RFA Engine

**"The 5-minute argument you need before every weekend build."**

RFA Engine runs structured adversarial AI debates on any idea. Six AI critics tear your idea apart. An Architect rebuilds it. You get a verdict: BUILD / PIVOT / KILL.

## Usage

```bash
# Full RFA (3-5 minutes)
python rfa-engine.py "your idea here"

# Quick Strike (30-60 seconds)
python rfa-engine.py "your idea here" --quick
```

## Setup

```bash
cp .env.example .env
# Add your ANTHROPIC_API_KEY to .env
pip install -r requirements.txt
```
